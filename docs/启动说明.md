# 快速启动指南

## 第一次运行

### 1. 安装依赖
```bash
npm install
```

### 2. 配置数据库

复制 `.env.example` 为 `.env` 并修改数据库配置：
```bash
cp .env.example .env
```

编辑 `.env` 文件，修改数据库密码：
```
DB_PASSWORD=你的MySQL密码
```

### 3. 创建数据库

在 MySQL 中执行：
```sql
CREATE DATABASE leixin_customer_service CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

然后导入数据库结构（参考项目根目录的 `数据库.md`）

### 4. 启动应用
```bash
npm run dev
```

## 日常使用

直接运行：
```bash
npm run dev
```

## 分别启动各个服务

如果需要单独调试某个服务：

```bash
# 终端1：启动后端API服务
npm run dev:server

# 终端2：启动前端开发服务器
npm run dev:react

# 终端3：启动Electron桌面应用
npm run dev:electron
```

## 构建生产版本

```bash
# 构建前端
npm run build

# 打包桌面应用
npm run package
```

打包后的应用在 `dist/` 目录中。

## 常见问题

### 问题1：数据库连接失败
- 检查 MySQL 服务是否启动
- 检查 `.env` 文件中的数据库配置是否正确
- 检查数据库是否已创建

### 问题2：端口被占用
- 修改 `.env` 文件中的 PORT 配置
- 或者关闭占用端口的程序

### 问题3：Electron 窗口打开失败
- 确保前端开发服务器已启动（http://localhost:5173）
- 等待几秒后再启动 Electron
