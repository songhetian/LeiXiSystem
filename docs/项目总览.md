# 客服管理系统 - 桌面版 🚀

## 项目简介

这是一个基于 **Electron + React + Fastify + MySQL** 技术栈开发的企业级客服管理桌面应用。采用浅绿色主题设计，提供客服人员管理、会话管理和质检管理等核心功能。

## ✨ 核心特性

### 🎨 设计特色
- **浅绿色主题** - 清新舒适的视觉体验
- **左右布局** - 左侧导航，右侧内容
- **现代化UI** - 圆角按钮，流畅动画，阴影效果
- **模态框交互** - 优雅的表单操作体验
- **Toast提醒** - 即时的操作反馈

### 💼 功能模块

#### 1. 客服人员管理
- ✅ 添加/编辑/删除客服
- ✅ 客服信息管理
- ✅ 评级系统（1-5星）
- ✅ 状态管理（在职/离职）

#### 2. 会话管理
- ✅ 会话列表查看
- ✅ 实时状态监控
- ✅ 会话详情展示
- ✅ 满意度评分

#### 3. 质检管理
- ✅ 多维度评分（服务态度、专业能力、沟通技巧、合规性）
- ✅ 自动计算综合得分
- ✅ 质检意见记录
- ✅ 质检历史查询

## 🚀 快速开始

### 方式一：一键启动（推荐）

**Windows:**
```bash
双击运行 start.bat
```

**Mac/Linux:**
```bash
chmod +x start.sh
./start.sh
```

### 方式二：手动启动

1. **安装依赖**
```bash
npm install
```

2. **配置数据库**
```bash
# 复制配置文件
cp .env.example .env

# 编辑 .env，修改数据库密码
```

3. **初始化数据库**
```bash
mysql -u root -p < database/init.sql
```

4. **启动应用**
```bash
npm run dev
```
