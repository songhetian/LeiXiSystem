## 🚀 快速开始

### 1. 安装依赖
```bash
cd customer-service-desktop
npm install
```

### 2. 配置数据库
```bash
# 复制配置文件
cp .env.example .env

# 编辑 .env，修改数据库密码
# 初始化数据库
mysql -u root -p < database/init.sql
```

### 3. 启动应用

**Windows:**
```bash
双击 start.bat
```

**Mac/Linux:**
```bash
chmod +x start.sh
./start.sh
```

**或者手动启动:**
```bash
npm run dev
```

## ⚠️ 注意事项

1. **首次运行前必须：**
   - 安装 Node.js >= 18.0.0
   - 安装 MySQL >= 8.0
   - 运行 `npm install`
   - 初始化数据库
   - 配置 .env 文件

2. **开发时注意：**
   - 修改后端代码需要重启服务
   - 修改前端代码会自动热重载
   - 修改 Electron 主进程需要重启应用

3. **数据库连接：**
   - 确保 MySQL 服务已启动
   - 检查 .env 配置是否正确
   - 确保数据库已创建

## 🐛 常见问题

### 问题 1：npm install 失败
```bash
npm cache clean --force
npm install
```

### 问题 2：数据库连接失败
- 检查 MySQL 服务是否启动
- 检查 .env 配置是否正确
- 检查数据库是否存在

### 问题 3：端口被占用
- 修改 .env 中的 PORT 配置
- 或者关闭占用端口的程序

### 问题 4：Electron 窗口无法打开
- 确保 Vite 服务器已启动
- 等待几秒后再启动 Electron

详细解决方案请查看 [INSTALL.md](./INSTALL.md)
