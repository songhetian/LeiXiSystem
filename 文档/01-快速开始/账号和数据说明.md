## ⚠️ 注意事项

### 密码安全
- ❌ 不要在生产环境使用这些测试账号
- ❌ 不要使用简单密码（123456）
- ✅ 生产环境必须修改默认密码
- ✅ 建议使用强密码策略

### 数据清理
- 测试完成后可以重置数据库
- 生产环境部署前删除测试数据
- 保留必要的基础数据（部门、职位等）

### 数据备份
```bash
# 备份当前数据库
mysqldump -u root -p leixin_customer_service > backup_$(date +%Y%m%d).sql

# 恢复数据库
mysql -u root -p leixin_customer_service < backup_20241109.sql
```
