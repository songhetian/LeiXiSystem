🚀 快速开始
========================================

1. 进入项目目录
   cd customer-service-desktop

2. 安装依赖
   npm install

3. 配置数据库
   - 复制 .env.example 为 .env
   - 修改数据库密码
   - 运行 database/init.sql

4. 启动应用
   Windows: 双击 start.bat
   Mac/Linux: ./start.sh
   或: npm run dev
