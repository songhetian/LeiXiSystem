## 🎯 立即体验

### 1. 刷新页面

按 **F5** 或 **Ctrl+R** 刷新浏览器

### 2. 进入权限管理

点击左侧菜单：**员工管理 > 权限管理**

### 3. 配置权限（超简单！）

#### 方式1：一键开启模块（推荐⭐）

```
1. 点击任意角色的"配置权限"按钮
2. 直接点击模块标题（如"员工管理"）
3. 整个模块的所有权限自动开启！
4. 再次点击可关闭
```

**示例**：
- 点击"员工管理" → 自动开启6个权限
- 点击"质检管理" → 自动开启3个权限
- 点击"考勤管理" → 自动开启4个权限

#### 方式2：精细控制

```
1. 展开模块
2. 勾选/取消具体权限
3. 自动保存
```

## 📝 快速上手

**需求**：新入职的客服专员

```
步骤：

2. 找到新员工

4. 勾选"客服专员"
5. 完成！
```

### 场景2：创建客服组长角色

**需求**：管理小组，有部分管理权限

```

   - 名称：客服组长
   - 级别：4

2. 点击"配置权限"

3. 一键开启需要的模块：
   - 点击"案例管理" ✓
   - 点击"质检管理" ✓
   - 点击"考勤管理" ✓

4. 完成！只需3次点击
```

### 场景3：调整角色权限

**需求**：客服专员需要创建案例的权限

```

2. 点击"配置权限"
3. 点击"案例管理"模块标题
4. 整个模块自动开启
5. 完成！
```

## 🎯 效率对比

### 配置一个完整角色的权限

**优化前**：
```
1. 打开权限对话框
2. 找到第一个模块
3. 逐个勾选权限（假设6个）
4. 找到第二个模块
5. 逐个勾选权限（假设4个）
6. 找到第三个模块
7. 逐个勾选权限（假设3个）
...
总计：需要点击约30-40次
耗时：约2-3分钟
```

**优化后**：
```
1. 打开权限对话框
2. 点击"员工管理"模块标题
3. 点击"质检管理"模块标题
4. 点击"考勤管理"模块标题
...
总计：需要点击约5-8次
耗时：约20-30秒
```

**效率提升：6倍！⚡**
