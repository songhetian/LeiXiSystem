## 🐛 遇到问题？

### 常见问题

**Q: 端口被占用**
```bash
# 查找占用端口的进程
netstat -ano | findstr :5173
netstat -ano | findstr :3001

# 结束进程
taskkill /PID <进程ID> /F
```

**Q: 数据库连接失败**
- 检查 `.env` 文件配置
- 确保 MySQL 服务已启动
- 验证用户名和密码

**Q: 页面空白**
- 清除浏览器缓存
- 检查控制台错误
- 确保后端服务已启动
