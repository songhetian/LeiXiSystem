## 🚀 使用指南

### 快速启动

```bash
# 1. 升级数据库（首次使用或升级）
upgrade-employee.bat

# 2. 启动服务
npm start

# 3. 访问系统
http://localhost:5173

# 4. 登录
用户名: admin
密码: admin123
```

### 功能使用

#### 员工管理
1. 点击侧边栏"员工管理" → "员工管理"
2. 查看所有员工列表（21个员工）
3. 使用搜索功能快速查找
4. 点击头像查看员工详情
5. 点击"编辑"修改员工信息
6. 点击"+ 添加员工"新增员工

#### 搜索功能
1. **关键词搜索**：输入姓名、工号、邮箱或手机号
2. **部门筛选**：选择特定部门
3. **状态筛选**：选择在职/停用/离职
4. **评级筛选**：选择1-5星
5. **清除筛选**：点击"清除筛选"按钮

#### 员工详情
1. 点击员工头像
2. 查看完整档案信息
3. 查看在职时长（自动计算）
4. 查看紧急联系人信息

#### 变动记录
1. 点击侧边栏"员工管理" → "变动记录"
2. 查看所有员工的变动历史
3. 使用筛选器查看特定类型
4. 查看本月统计数据

## 🔍 搜索功能详解

### 关键词搜索
**支持字段：**
- 员工姓名（模糊匹配）
- 员工工号（模糊匹配）
- 邮箱地址（模糊匹配）
- 手机号码（模糊匹配）

**使用示例：**
- 输入"张" → 找到所有姓张的员工
- 输入"E001" → 找到工号E001的员工
- 输入"138" → 找到手机号包含138的员工
- 输入"@company" → 找到公司邮箱的员工

### 部门筛选
**功能：**
- 下拉选择部门
- 只显示该部门的员工
- 选择"全部部门"显示所有员工

**使用场景：**
- 查看技术部有多少人
- 统计销售部员工
- 管理客服部人员

### 状态筛选
**选项：**
- 全部状态
- 在职（active）
- 停用（inactive）
- 离职（resigned）

**使用场景：**
- 查看在职员工数量
- 统计离职员工
- 管理停用账号

### 评级筛选
**选项：**
- 全部评级
- 5星 ⭐⭐⭐⭐⭐
- 4星 ⭐⭐⭐⭐
- 3星 ⭐⭐⭐
- 2星 ⭐⭐
- 1星 ⭐

**使用场景：**
- 查找优秀员工（5星）
- 查找需要培训的员工（1-2星）
- 统计各评级人数

### 组合搜索
**示例1：**
```
部门：技术部
状态：在职
评级：5星
→ 查找技术部的优秀在职员工
```

**示例2：**
```
关键词：张
部门：销售部
状态：在职
→ 查找销售部姓张的在职员工
```

**示例3：**
```
状态：离职
评级：5星
→ 查找已离职的优秀员工（可能需要挽留）
```

- 恢复显示所有员工
