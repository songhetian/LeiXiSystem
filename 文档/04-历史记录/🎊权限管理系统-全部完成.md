## 💡 使用场景

### 场景1：新员工入职流程

```
1. 用户注册 → 状态"pending"
2. 管理员审核 → 进入"员工审核"
3. 审核通过 → 状态变为"active"
4. 分配角色 → 在"员工管理"中编辑员工
5. 选择角色 → 如"客服专员"
6. 完成！员工可以登录并使用系统
```

### 场景2：查看本月新入职员工

```
员工管理：
1. 入职日期：2024-11-01 至 2024-11-30
2. 部门：客服部
3. 状态：在职
4. 查看结果 → 本月入职的在职客服
```

### 场景3：批量为新员工分配角色

```
权限管理 > 用户角色：
1. 注册日期：选择本周
2. 部门：客服部
3. 勾选所有新员工
4. 批量分配"客服专员"角色
5. 完成！
```

### 场景4：查看本周的员工变动

```
变更记录：
1. 变动日期：2024-11-04 至 2024-11-09
2. 变动类型：全部类型
3. 查看结果 → 本周所有变动记录
```
